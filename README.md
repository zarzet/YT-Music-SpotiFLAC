# SpotiFLAC‑YT‑Music‑Extension

##### A high‑performance SpotiFLAC extension that integrates YouTube Music as a source for search, metadata, and high‑quality audio downloads.

## Features
- Custom search with multi‑stage query logic  
- Advanced track matching (title, artist, album, duration scoring)  
- Quality‑aware downloads (AAC 256, Opus 160, AAC 128)  
- Optimized performance with caching and rate‑limit backoff  
- Fully compatible with the SpotiFLAC 3.x extension system  

## Installation
1. Download the latest .spotiflac-ext file from the Releases page  
2. Open SpotiFLAC and navigate to Extensions  
3. Import the file  
4. Enable the extension  

## Development

`
npm install
npm run build
`

The built extension appears in the dist/ directory.
